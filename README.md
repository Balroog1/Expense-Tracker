# 💰 Personal Expense Tracker

A command-line Personal Expense Tracker built with Python.

---

## 🚀 Features

### ✅ Current Features

- Interactive command-line menu
- Add new expenses
- Input validation
- View all expenses
- Expense summary report
  - Total expenses
  - Number of expenses
  - Highest expense
  - Lowest expense
  - Average expense
- In-memory expense storage
